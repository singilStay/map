How to Use the Boiler & 보일러 사용방법

To check the heating, look for the phrase 'Heating', and to check the hot water, look for the phrase 'Hot Water = 온수'

💧Hot water must be set at 60 degrees🌡

检查加热时，查找“Heating”字样；检查热水时，查找“Hot Water = 온수”字样。
💧热水必须设置为60度🌡

난방을 사용하시려면 '난방' 버튼을 눌러주세요,
💧온수는 60도에 설정되어야 합니다🌡


How to Set Indoor Heating
- Press the heating button and adjust the dial to set your desired indoor temperature.

如何设置室内加热 按下加热按钮，
- 并调整旋钮设置您希望的室内温度。

실내 난방 설정 방법
- 난방 버튼을 눌러 다이얼을 조정하여 원하는 실내 온도를 설정하세요.


If the room does not warm up even when the heating mode is on, please check the ondol temperature setting.
- If the ondol temperature is set to 75°C, press the heating button again.

如果即使在开启加热模式时房间仍未变暖，请检查温控设置。
- 如果温控设置为75°C，请再次按下加热按钮。

난방 모드가 켜져 있는데도 방이 따뜻해지지 않으면 온돌 온도 설정을 확인하세요.
- 온돌 온도가 75°C로 설정되어 있으면, 난방 버튼을 다시 눌러주세요.
